# Libyan Safe Residence — Official Website

Arabic-first public website for **شركة الإقامة الآمنة الليبية للخدمات العمالية / Libyan Safe Residence**.

This is a standalone website repo created from the PDF/company-profile source of truth.

## Source of truth

- `docs/lsr-pdf-source-of-truth.md`

## Commands

```bash
npm install
npm run lint
npm run typecheck
npm test
npm run build
```

## Scope

Built as a public-facing website, not the old internal demo:

- Hero
- Who we are
- Services
- Workflow
- Sectors served
- Why LSR
- Contact
